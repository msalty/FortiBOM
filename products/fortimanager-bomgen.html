VM Subscription Bundle Configuration

Step 2
Best value for VM deployments. VM Subscription bundles include FortiManager VM S-series + FortiCare Premium in one SKU (FCx-10-FMGVS-258-01-DD). Fully stackable — order multiple tiers to reach any device count. To upgrade to FortiCare Elite, add the Elite Upgrade SKU.
How to size: Enter your total device count and the tool will pack it using the largest available tiers (1,000 → 100 → 10) to minimize SKU line items. All SKUs use co-term -DD suffix. Each VM subscription includes 5 ADOMs; add the ADOM Subscription License for more.
